# Rwanda Crop Yield Prediction - Linear Regression Summative

**Mission:** Smallholder farmers in Rwanda lack data-driven tools to estimate crop yields before harvest. This project builds a multivariate regression model predicting crop yield (hg/ha) from agronomic inputs including area harvested, production trends, crop type, and year - empowering agricultural extension officers and farmers to make informed decisions that improve food security across Rwanda.

**Dataset:** FAOSTAT Rwanda Crops and Livestock Products (1961-2024), 35 crop types
**Target Variable:** yield - crop yield in hectograms per hectare (hg/ha)
